Nom du projet: Pierre Feuille Ciseaux version The Big Bang Theory

Auteur: M. BECQUER Michaël 

État du projet: Fini

Rapide description : le jeu classique du pierre feuille ciseaux, avec l'amélioration The Big Bang Theory qui inclut 
les possibilités Lézard, Spock et une fonctionalités cachées !

Règle du Pierre Feuille Ciseaux Lézard Spock

- les ciseaux coupent la feuille
- la feuille recouvre la pierre
- la pierre écrase le lézard
- le lézard empoisonne spock
- spock écrabouille les ciseaux
- les ciseaux décapitent le lézard
- le lézard mange la feuille
- la feuille désavoue spock
- spock vaporise la pierre
et comme toujours la pierre casse les ciseaux ...
